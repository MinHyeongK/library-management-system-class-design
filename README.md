# 도서관 관리 시스템(Library Management System)
도서관 관리 시스템의 클래스 설계 이후 설계한 클래스를 스프링으로 변환해보는 작업을 연습하는 프로젝트입니다.

p.s. 해당 프로젝트는 스터디의 첫 시작이라 LLM(GPT-5)의 도움을 받아 작성하였습니다. 2번쨰 프로젝트부터는 자력으로 작성할 예정입니다.
